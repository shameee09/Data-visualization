# Placement Analytics Dashboard 

## Overview
This dashboard was developed using Google Data Studio to analyze student placement performance, salary trends, and specialization-wise insights.

## Features
- Placement Rate Analysis
- Average Salary Analysis
- Salary by Specialisation
- MBA Percentage vs Salary
- Interactive Filters

## Tools Used
- Python
- Pandas
- NumPy
- Google Data Studio
